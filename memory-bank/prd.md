# 产品需求文档（PRD）

## 项目名称
Heartbeat — ECG 心电信号分类与公平性分析系统

## 项目目标
构建一个端到端的 ECG 信号处理系统，实现：
1. 从 MIT-BIH 心律失常数据库读取真实心电数据
2. 信号预处理（小波去噪）
3. 心拍自动分类（5 类：N 正常, A 房性早搏, V 室性早搏, L 左束支传导阻滞, R 右束支传导阻滞）
4. 在分类过程中引入公平性约束，消除年龄偏见
5. 通过 MATLAB 可视化去噪后的信号
6. 支持实时串口数据采集（可选）

## 功能需求

### 核心功能
- **数据读取**：从 MIT-BIH 数据库读取 48 条记录，提取 MLII 导联信号
- **信号去噪**：使用 db5 小波对 ECG 信号进行 9 层分解 + 软阈值去噪
- **心拍分割**：以 R 峰为中心，截取前 100 + 后 200 = 300 个采样点
- **公平性处理**：
  - 预处理阶段：AIF360 Reweighing 重加权
  - 训练阶段：Adversarial Debiasing 对抗去偏
- **模型训练**：RNN 分类器，支持 5 类心拍识别
- **公平性评估**：Statistical Parity、Disparate Impact、Equal Opportunity 等指标

### 辅助功能
- UDP 数据传输到 MATLAB 进行可视化
- 串口采集实时 ECG 数据（6 导联）
- 模拟 ECG 数据生成（用于测试）

## 非功能需求
- 代码模块化，每个文件不超过 300 行
- 支持 Windows 环境运行
- 数据集路径可配置
