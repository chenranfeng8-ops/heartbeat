## 二、引言

### 2.1 产品背景
Heartbeat 用于对心电信号（ECG）进行自动处理和分析。系统支持从 MIT-BIH 心律失常数据库读取数据，完成信号去噪、心拍分割、分类预测，并提供公平性评估能力，减少模型在年龄维度上的偏差。

### 2.2 适用范围
- 离线 ECG 数据分析与实验验证
- 心律失常五分类研究（N/A/V/L/R）
- AI 公平性评估与去偏实验
- MATLAB 联动可视化（可选）

### 2.3 目标用户
- 医工交叉研究人员
- 算法工程师与数据科学人员
- 需要进行 ECG 模型训练与公平性评估的教学/科研团队

### 2.4 术语解释（去技术化）
- ECG：心电图信号。
- R 峰：每次心跳里最明显的尖峰，用来对齐心拍。
- 心拍分割：从长信号中切出每次心跳的固定长度片段。
- 小波去噪：用于“清理毛刺和干扰”的信号清洗步骤。
- Reweighing：给样本重新分配权重，降低偏见。
- Adversarial Debiasing：训练时加一个“纠偏器”，让模型更公平。

## 四、详细操作指南

### 4.1 数据读取模块
操作步骤：
1. 系统按记录号依次读取输入数据。
2. 自动选择 MLII 导联。
3. 读取注释文件 atr 定位 R 峰。
4. 跳过异常记录，如缺失文件、导联缺失、长度不足。


- 控制台出现“记录不存在/缺失”提示但程序继续运行，属于容错机制。

### 4.2 信号预处理模块
操作步骤：
1. 系统对心电片段执行 db5 小波分解。
2. 自动计算阈值并进行软阈值处理。
3. 重构出更平滑的去噪信号。

：
- 输出阈值和去噪后的信号范围信息。

### 4.3 心拍分割与标签生成
操作步骤：
1. 以 R 峰为中心切片：前 100 点 + 后 200 点。
2. 每个心拍片段长度固定为 300。
3. 将标签映射到五分类：N、A、V、L、R。


- 最终样本形状为 样本数 x 300 x 1。

### 4.4 公平性处理
操作步骤：
1. 将年龄转成二值属性：`age >= 30` 记为 1，`age < 30` 记为 0。
2. 先执行 Reweighing 预处理。
3. 再进入对抗去偏训练 Adversarial Debiasing。


- 控制台出现公平性指标输出，如差异值、影响比。

### 4.5 训练与评估模块
操作步骤：
1. 按比例划分训练集和测试集，测试比例 0.2。
2. 训练分类模型并输出精度指标。
3. 对比“去偏前/去偏后”公平性结果。


- 出现 Accuracy、Balanced Accuracy、Disparate Impact 等指标。


### 4.6 UDP/串口
操作步骤：
1. 配置串口参数与接收端地址。
2. 将处理后数据按分片通过 UDP 发送。
3. 在接收端完成拼接和展示。
4. 启动 MATLAB 引擎。
5. 将多导联数据传入 MATLAB。
6. 以 6 子图方式显示波形。


- 接收端持续收到数据包，MATLAB 弹出图窗并显示 6 条导联曲线。

## 五、常见问题与故障排除

### 5.1 依赖安装失败
问题现象：`ModuleNotFoundError`。
处理方法：
1. 确认当前 Python 环境与运行环境一致。
2. 重新安装缺失包（例如 `pip install wfdb`、`pip install aif360`）。
3. 使用 `pip list` 检查安装结果。

### 5.2 MATLAB 引擎无法导入
问题现象：`No module named matlab.engine`。
处理方法：
1. 确认 MATLAB 已安装。
2. 在 MATLAB 对应 Python 环境中安装引擎。
3. 若本次只做算法验证，可先注释 MATLAB 可视化相关代码。

### 5.3 数据文件缺失
问题现象：提示 `.hea` 文件缺失或记录无法读取。
处理方法：
1. 检查数据目录是否完整。
2. 确认记录号文件齐全（hea/dat/atr）。
3. 校验路径拼写与大小写。

### 5.4 运行报参数错误
问题现象：`TypeError: pre() takes 0 positional arguments but 1 was given`。
处理方法：
1. 检查主程序中 `pre` 调用方式是否与函数定义一致。
2. 保持 `pre()` 无参调用，或按设计重构函数签名后再调用。
3. 调整后重新运行验证。

### 5.5 出现 NaN 或训练不稳定
问题现象：日志中提示 NaN。
处理方法：
1. 检查输入数据是否有空值。
2. 检查去噪阈值是否异常。
3. 先缩小数据规模做最小样例验证。

### 5.6 公平性指标无改善
问题现象：去偏后指标变化不明显。
处理方法：
1. 检查年龄分组是否正确（阈值 30）。
2. 提高样本数量，避免分组极不平衡。
3. 调整对抗损失权重后重新训练。
